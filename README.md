Project: Titanic Dataset Data Visualization

Tools Used:
- Python
- Pandas
- Matplotlib
- Google Colab

Visualizations:
1. Survival Distribution
2. Gender Distribution
3. Passenger Class Distribution
4. Age Distribution
5. Survival by Gender
6. Survival by Passenger Class

Key Findings:
- Female passengers had higher survival rates.
- Most passengers traveled in 3rd class.
- Passenger class influenced survival chances.
